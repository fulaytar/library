# Library App

A simple web-based library application to manage books. Users can **view, add, edit, delete, and export** book details.

## Features

- **View books**: See a paginated list of all books with details.
- **Add book**: Open a modal form to add a new book (Title, Author, Year, Genre, Pages).
- **Edit book**: Edit existing book details via modal form.
- **Delete book**: Remove books with confirmation.
- **Export book**: Copy book details to clipboard (one book at a time).
- **Search & Filter**: Search books by title, author, or year.

## Technologies Used

- **JavaScript (ES6+)**
- **HTML & CSS**
- **Bootstrap 5**
- **Modular MVC pattern**: Separate Model, View, and Controller
